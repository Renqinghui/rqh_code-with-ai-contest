# 5G 路测数据可视化看板

## 📊 项目简介

5G 路测数据可视化看板是一个基于 Python 和 Streamlit 构建的交互式 Web 应用，专门用于分析和可视化 G 网络路测数据。本应用能够将原始的 5G 路测数据快速转换为直观的可视化图表和地图，帮助网络优化工程师实时监控网络覆盖质量、信号强度分布和基站性能。

## 🎯 核心功能

### 🗺️ 离线交互式地图
- **2D 散点地图**：根据 RSRP 信号强度动态着色（绿色＞-90dBm，红色＜-110dBm）
- **3D 柱状地图**：信号点以 3D 柱状图形式显示，高度随信号质量变化
- **信号热力图**：展示信号强度的密度分布
- **完全离线运行**：无需网络连接，不依赖在线地图服务

### 📈 智能数据筛选
- **频段筛选**：多选频段（n28, n41, n78 等）
- **RSRP 范围筛选**：滑动条选择信号强度范围
- **SINR 阈值筛选**：设置最小信噪比
- **终端类型筛选**：按设备类型筛选（Smartphone, CPE, IoT）
- **实时联动更新**：所有筛选器实时更新地图和图表

### 📊 数据分析与可视化
- **各频段基站统计**：柱状图展示各频段基站数量分布
- **信号强度分布**：直方图展示 RSRP 值分布
- **终端类型分析**：饼图展示不同类型终端占比
- **关键性能指标**：实时计算并显示平均 RSRP、平均 SINR、基站总数等
- **信号质量统计**：统计信号优秀、良好、较差的比例

### 💾 数据管理
- **智能数据加载**：自动检测 CSV 文件，标准化列名
- **数据预览**：侧边栏显示数据样本
- **数据导出**：支持导出筛选后的数据为 CSV 格式
- **容错处理**：数据缺失时自动生成演示数据

## 🚀 快速开始

### 1. 环境要求
- Python 3.8+
- pip 包管理器

### 2. 安装步骤

#### 克隆项目
```bash
git clone <项目仓库地址>
cd 5g-offline-dashboard
```

#### 安装依赖
```bash
pip install -r requirements.txt
```

**requirements.txt 内容**：
```
streamlit>=1.28.0
pandas>=2.0.0
numpy>=1.24.0
seaborn>=0.12.0
matplotlib>=3.7.0
pydeck>=0.8.0
```

### 3. 准备数据文件

将您的 5G 路测数据文件命名为 \`signal_samples.csv\`，并放置在以下任一位置：

1. \`./data/signal_samples.csv\`（推荐）
2. \`./signal_samples.csv\`
3. \`signal_samples.csv\`（与主程序同一目录）

**数据格式要求**：
CSV 文件应包含以下列（不区分大小写，会自动标准化）：
- \`Latitude\` 或 \`lat\` - 纬度
- \`Longitude\` 或 \`lon\` 或 \`lng\` - 经度
- \`Cell_ID\` 或 \`CellID\` - 小区ID
- \`Band\` - 频段
- \`RSRP_dBm\` 或 \`RSRP\` - 信号强度
- \`SINR_dB\` 或 \`SINR\` - 信噪比
- \`TerminalType\` - 终端类型（可选）
- \`Download_Mbps\` - 下载速率（可选）

### 4. 运行应用

```bash
streamlit run 5g_offline_map_dashboard.py
```

应用将在浏览器中自动打开，访问地址：http://localhost:8501

## 🔧 地图操作指南

### 交互操作
1. **缩放**：鼠标滚轮或工具栏缩放按钮
2. **平移**：鼠标拖动地图
3. **旋转**：按住 Ctrl 键 + 鼠标拖动
4. **倾斜视角**：按住 Shift 键 + 鼠标拖动
5. **点击查看详情**：点击地图上的点查看详细信息

### 颜色说明
- 🟢 **绿色**：信号优秀 (RSRP > -90dBm)
- 🟡 **黄色**：信号良好 (-110dBm ≤ RSRP ≤ -90dBm)
- 🔴 **红色**：信号较差 (RSRP < -110dBm)

## 🎮 使用示例

### 基本使用流程
1. **启动应用**：\`streamlit run 5g_offline_map_dashboard.py\`
2. **加载数据**：应用自动检测并加载数据文件
3. **查看概览**：查看关键指标和地图
4. **筛选数据**：在侧边栏设置筛选条件
5. **分析结果**：查看筛选后的可视化结果
6. **导出数据**：如需要，导出筛选后的数据

### 网络优化分析示例
1. 筛选 RSRP 范围 [-120, -100] 查看弱覆盖区域
2. 选择特定频段（如 n78）分析高频频段覆盖
3. 查看 SINR 分布，定位干扰问题
4. 导出问题区域数据用于现场优化

## 🔍 故障排除

### 常见问题

1. **文件未找到错误**
   ```
   错误：未找到数据文件 signal_samples.csv
   解决：确保文件放在正确位置，或检查文件权限
   ```

2. **依赖安装失败**
   ```
   错误：无法安装某些包
   解决：更新pip：pip install --upgrade pip
   或使用虚拟环境
   ```

3. **内存不足**
   ```
   错误：内存不足
   解决：减少数据采样比例，或增加系统内存
   ```

4. **端口被占用**
   ```
   错误：端口8501已被占用
   解决：使用其他端口：streamlit run app.py --server.port=8502
   ```
